# LUG @ JLU 互助书柜

吉林大学 Linux 用户协会互助书柜的前端页面。

- `index.html` 前端渲染的独立页面
    - `style.css` 手搓的简易样式表
    - `vue.global.prod.js` Vue.js 3
    - `script.js` Vue 渲染逻辑
    - `paul-*.jpg` 背景图片

- `csv2json.py` 将 Google 表格提供的 CSV 转换成 JSON
- `update.sh` 用于获取表格信息的脚本
